Đồ án thực hiện phân tích cú pháp cấu trúc ngữ đoạn trong tiếng Việt.<br/>
Sử dụng thuật toán Maxinum Matching để tách từ. Sau đó dùng thuật toán CKY để phân tích cú pháp.
